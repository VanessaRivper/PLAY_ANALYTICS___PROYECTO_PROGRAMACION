🎮Dataset:

  El Dataset utilizado en este proyecto fue obtenido de Kaggle:
  https://www.kaggle.com/datasets/bhushandivekar/video-game-sales-and-industry-data-1980-2024

  Este Dataset contiene información de miles de Videojuegos desde 1980 hasta 2024. Incluye datos sobre ventas, géneros y evaluaciones.

🎮Variables principales:
  - Título del juego
  - Plataforma
  - Género
  - Editor
  - Desarrollador del juego
  - Puntuación crítica
  - Ventas gobales
  - Fecha de lanzamiento

  El dataset permite analizar las tendencias en la industria de los Videojuegos y responder preguntas relacionadas con la popularidad, ventas y      características de los juegos.

🎮Objetivo:

  Desarrollar un programa en consola que permita explorar el Dataset de videojuegos mediante un menú interactivo, facilitando la búsqueda y el       análisis básico de la información.

🎮Funciones del programa:

  Permite buscar registros a partir de un término ingresado por el usuario, incluye funcionalidades estadísticas básicas y opciones de ____

🎮Cómo correr el programa:
  1. Descargar el repositorio.
  2. Abrir la carpeta en Visual Studio Code.
  3. Ejecutar el archivo main.py.
  4. Usar el menú en consola.
